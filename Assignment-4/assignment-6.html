Q.6 What is template literals in ES6 and how do you use them? answer:- Template literals, also known as template strings, are a new feature introduced in ECMAScript 6 (ES6) that allows developers to create more expressive and flexible strings in JavaScript. Template literals are enclosed in backticks (``) instead of single or double quotes. Template literals provide the following advantages over traditional strings: String Interpolation: Template literals allow easy embedding of variables and expressions within the string using ${}. This is called string interpolation and makes it simpler to include dynamic values in the string. Multi-line Strings: Template literals can span multiple lines without the need for escape characters or concatenation, making multiline strings more readable. Expression Evaluation: The expressions inside ${} are evaluated, and their results are included in the string. This allows for more complex string construction and dynamic content.
