Q.10 What is the purpose of the spread operator (...) in ES6? answer:- The spread operator (...) in ECMAScript 6 (ES6) is a powerful feature that serves multiple purposes and allows you to perform various operations in a concise and flexible way. Spread in Arrays: Copying Arrays: The spread operator can be used to create a shallow copy of an array. It allows you to copy the elements of one array into a new array without modifying the original array. Concatenating Arrays: The spread operator can be used to concatenate multiple arrays into a new array. Passing Arguments: The spread operator can be used to pass the elements of an array as individual arguments to a function. Examples: javascript // Copying Arrays const originalArray = [1, 2, 3]; const copiedArray = [...originalArray]; // Concatenating Arrays const array1 = [1, 2]; const array2 = [3, 4]; const concatenatedArray = [...array1, ...array2]; // Passing Arguments to Function function sum(a, b, c) { return a + b + c; } const numbers = [1, 2, 3]; const result = sum(...numbers); // Equivalent to sum(1, 2, 3) Spread in Objects: Copying Objects: The spread operator can be used to create a shallow copy of an object. It allows you to copy the properties of one object into a new object without modifying the original object. Merging Objects: The spread operator can be used to merge multiple objects into a new object. Examples: javascript // Copying Objects const originalObj = { name: 'John', age: 30 }; const copiedObj = { ...originalObj }; // Merging Objects const obj1 = { name: 'John' }; const obj2 = { age: 30 }; const mergedObj = { ...obj1, ...obj2 }; Spread in Function Arguments: The spread operator can be used to pass multiple arguments to a function when the function expects separate arguments instead of an array. Example: javascript function sum(a, b, c) { return a + b + c; } const numbers = [1, 2, 3]; const result = sum(...numbers); // Equivalent to sum(1, 2, 3) Spread with Other Iterables: The spread operator can also be used with other iterables such as strings, sets, and maps to create arrays or objects. Examples: javascript // String to Array const str = 'hello'; const chars = [...str]; // Equivalent to ['h', 'e', 'l', 'l', 'o'] // Set to Array const set = new Set([1, 2, 3]); const arrFromSet = [...set]; // Equivalent to [1, 2, 3] // Map to Array of Objects const map = new Map([[1, 'one'], [2, 'two']]); const arrOfObjects = [...map]; // Equivalent to [ [1, 'one'], [2, 'two'] ] In summary, the spread operator ... in ES6 is a versatile feature that simplifies copying, merging, and passing elements of arrays, properties of objects, or other iterables in a more concise and flexible way. It greatly enhances the readability and efficiency of JavaScript code.
