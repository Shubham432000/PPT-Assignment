Q.4 Whats difference between Inline, Inline Block and block ? Answer:- Inline, inline-block, and block are display property values in CSS that determine how elements are rendered and positioned in relation to other elements. Here's the difference between them: Inline: Elements with the display property set to "inline" are rendered in a line, one after another, without starting a new line. They do not create line breaks and only take up the necessary width and height to contain their content. Examples of inline elements are , , and . Inline elements cannot have fixed width or height, and margin and padding properties only affect the left and right sides, not top and bottom. Inline elements cannot contain block-level elements. They respect width and height settings and can have margin and padding applied on all sides. Inline-block elements are placed horizontally in a line but can have vertical alignment.
